# ★★★ 최우선 과제 — 자동공고 생성기에서 회사 상호명 빼기 (반복 재발 중)

> 대표님 방침: **모든 구인공고에 회사 상호명(해광 화물운송)을 넣지 않는다. 모집내용만.**
> 현재 `haegwang-auto-job-draft`(자동공고 생성) 작업이 새 공고를 만들 때마다 상호명을 다시 넣어서, 매번 수동으로 지워야 하는 상태. **생성기 자체를 고쳐야 근본 해결.**

## 무엇이 잘못 들어가나 (새 공고마다 재발)
1. 공고 카드에 `<div class="job-company">해광 화물운송</div>` ← 회사명 채워서 생성됨
2. 공고 상세(job-desc) 끝에 `※ 본 공고는 해광 화물운송이 직접 모집하는 공고입니다.` 문구
3. (JS) JobPosting 구조화데이터 hiringOrganization name 을 '해광 화물운송'으로

## 생성기에서 이렇게 바꿀 것 (집 PC: haegwang-auto-job-draft 작업 프롬프트 수정)
- 새 공고 카드의 `job-company`는 **비워서 생성**(`<div class="job-company"></div>`).
- job-desc에 **"※ 본 공고는 해광 화물운송이 직접 모집…" 문구 넣지 말 것.**
- 구조화데이터 회사명은 **'화물구인구직'** 으로.
- 지원 연락처(전화 010-7233-3916 / 이력서폼 https://forms.gle/HixdqkYMSfwD1kyH7)는 **유지**(상호명만 제외, 모집내용·지원방법은 유지).
- index.html에는 이미 `.job-company{display:none}` CSS가 있어 화면상 회사명줄은 숨겨지지만, **source·구조화데이터·상세문구에는 남으므로** 생성기에서 안 넣는 게 정답.
