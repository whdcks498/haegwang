# 유가 자동갱신 클라우드 이관 가이드 (투데이 유류정보 회귀 영구 해결)
**작성: 해광AI비서 | 2026-06-15 | 실행: 내일 회사 PC에서**

## 왜 하나
회사 PC의 유가 자동갱신이 매일 아침(약 07:33) `diesel-price.html`을 **옛 디자인으로 통째 재생성**해서, "투데이 유류정보" 리디자인이 계속 되돌아갑니다. 이를 **GitHub Actions(클라우드)로 옮겨**, 유가 데이터(fuel-prices.json)만 자동 갱신하고 페이지(diesel-price.html)는 건드리지 않게 만들면 영구 해결됩니다.

> 오늘 diesel-price.html은 일단 정상 버전으로 복구·배포했습니다. 단, 회사 자동갱신을 중단하기 전까지는 내일 아침 다시 덮일 수 있습니다.

---

## 내일 회사에서 할 일 (대표님)

### ① 회사 유가 갱신 프로그램 찾기 + 오피넷 키 확인
- 매일 아침 자동으로 도는 **유가 갱신 스크립트/프로그램**을 찾습니다. 단서:
  - Windows **작업 스케줄러**(taskschd.msc)에 등록된 작업, 또는 **시작프로그램**, 바탕화면/폴더의 `.py`·`.bat` 파일.
  - 커밋 메시지 "Update fuel prices (auto)"를 만드는 것.
- 그 스크립트 파일을 열어 **오피넷 API 키**를 찾습니다. 보통 `code=` 뒤의 문자열, 또는 `KEY=`, `apikey=` 형태(영문+숫자).
- **이 키 값을 복사해서 해광AI비서에게 전달**해 주세요. (이게 핵심)

### ② 회사 자동갱신 중단
- 위 작업 스케줄러 작업을 **사용 안 함(비활성화)** 처리 → 더 이상 diesel-price.html을 덮지 않게.
- (키만 알려주시면 중단은 천천히 하셔도 됩니다. 단 중단 전엔 페이지가 또 덮일 수 있음.)

1. **GitHub Actions 유가갱신 워크플로** 추가 — 매일 오피넷 무료 API에서 전국·시도별 유가를 받아 `fuel-prices.json`만 갱신. (키는 GitHub 비공개 Secret `OPINET_KEY`로 안전 보관)
2. **diesel-price.html 개선** — 4유종 카드뿐 아니라 시도별 가격표도 `fuel-prices.json`에서 읽도록(json만 바뀌면 페이지 전체가 자동 최신, 페이지 파일은 영구 고정).
3. 배포 + 검증.

## 오피넷 무료 API 참고 (해광AI비서용)
- 전국 평균(유종별): `avgAllPrice.do?out=json&code=KEY`
- 시도별 평균: `sidoPrice.do?out=json&code=KEY&prodcd=D047`(경유)
- 유종코드: 휘발유 B027 · 경유 D047 · 고급휘발유 B034 · LPG(부탄) K015
- 무료 API에 "전국 평균가격(현재)", "시도별 평균가격(현재)" 모두 제공 확인됨.
